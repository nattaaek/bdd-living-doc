## Getting Started

Ready to bring your BDD documentation to life? Let's get started! ğŸˆ

### Prerequisites

Make sure you have the following installed:

- **Node.js** (v14 or higher)
- **npm** or **yarn**

### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/yourusername/bdd-living-doc.git
   cd bdd-living-doc
   ```

2. **Install dependencies**:

   ```bash
   npm install
   # or
   yarn
   ```

### Running the Project

1. **Start the development server**:

   ```bash
   npm run dev
   # or
   yarn dev
   ```

2. **Open your browser** and navigate to `http://localhost:3000` to see the application in action!

## Usage

- **View Features**: The homepage displays a list of all BDD features.
- **Expand Scenarios**: Click on a feature to reveal its scenarios.
